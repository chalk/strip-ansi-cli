# strip-ansi-cli

> Strip [ANSI escape codes](https://en.wikipedia.org/wiki/ANSI_escape_code)

## Install

```
$ npm install --global strip-ansi-cli
```

## Usage

```
$ strip-ansi --help

  Usage
    $ strip-ansi <string>
    $ echo <string> | strip-ansi

  Example
    $ ls --color | strip-ansi
```
